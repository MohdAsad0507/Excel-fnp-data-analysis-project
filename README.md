# 🌸 Sales Analysis Dashboard – FNP (Excel Project)

## 📌 Project Overview

This project presents a comprehensive Sales Analysis Dashboard for Ferns N Petals (FNP), one of India’s leading online gifting and floral platforms.

The objective was to transform raw transactional data into actionable business insights using Microsoft Excel by leveraging Power Query, data modeling, and interactive dashboards.

---

## 📂 Dataset

* The dataset used in this project is a simulated sales dataset inspired by real-world operations of an online gifting platform (FNP)
* It includes:

  * Order details
  * Product categories
  * Customer information
  * Delivery timelines
* Data has been structured to reflect realistic business scenarios for analysis and visualization

---


## 🏗️ Project Workflow & Architecture
```
Raw Data (Folder)
      ↓
Power Query (Data Extraction & Transformation)
      ↓
Data Modeling (Excel Tables & Relationships)
      ↓
Pivot Tables & Measures
      ↓
Interactive Dashboard (Charts + KPIs + Slicers)
```


## 🧹 Data Preparation & Transformation

* Extracted raw data from multiple files/folders
* Cleaned and transformed data using Power Query Editor:

  * Removed duplicates & null values
  * Standardized formats (dates, currency, categories)
  * Created calculated columns (e.g., delivery difference)
* Loaded clean data into Excel data model

---

## 🧠 Data Modeling & Analysis

* Designed structured tables for efficient analysis
* Created relationships between datasets
* Built Pivot Tables for aggregation and slicing
* Used Excel functions and measures for:

  * Revenue calculations
  * Average metrics
  * Time-based analysis

---

## 📊 Dashboard Overview

The dashboard provides a 360° view of sales performance, featuring:

### 🔑 Key Performance Indicators (KPIs)

* Total Orders: 1000
* Total Revenue: ₹35,20,984
* Average Delivery Time: 5.53 days
* Average Customer Spend: ₹3,520.98

---

## 📈 Insights

### 🎉 Revenue by Occasions

* Highest revenue generated during Anniversary & Raksha Bandhan
* Seasonal spikes indicate strong event-driven demand

### 🧁 Revenue by Category

* Colors & Sweets dominate revenue contribution
* Opportunity to optimize underperforming categories

### 🌆 Top Cities by Orders

* Cities like Imphal, Dhanbad, Kavali show high order volume
* Helps identify strong regional markets

### 📅 Monthly Revenue Trends

* Clear seasonality patterns
* Revenue peaks during festive months

### 🛍️ Top Products by Revenue

* Products like Magnam Set & Quia Gift are top performers
* Useful for inventory and marketing focus

### ⏰ Revenue by Order Time

* Identifies peak ordering hours
* Can be used for targeted promotions
